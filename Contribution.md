## Manual Installaion
- Install nodejs in your system
- Install pnpm in your system
- git clone the repo
- Create the env file
- Run |
        cd packages/apps/db
        npx prisma generate
- RUN pnpm run build
- RUN pnpm run start

---
Ports running
3000: Web App
3001: Landing Page
4000: Express backend
